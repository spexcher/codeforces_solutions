That is, $$$a_j$$$ is decreased by $$$a_i$$$. Note that fighter $$$j$$$'s rating can become negative. 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each testcase, output a single integer — the maximum rating that the last remaining fighter can
